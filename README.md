# 🎯 秋招简历制作工具

一个专为秋招设计的交互式简历制作网站，支持多版本管理、实时预览、PDF导出等功能。
![Uploading image.png…]()

## ✨ 核心功能

- **📝 交互式编辑**：可视化表单输入，实时预览效果
- **🔄 多版本管理**：快速保存和切换不同岗位的简历版本
- **📄 PDF导出**：一键导出高质量PDF，格式完美保真
- **💾 数据备份**：支持导出/导入备份，防止数据丢失
- **📱 响应式设计**：支持桌面端和移动端访问

## 🚀 快速开始

### 本地开发

1. **安装依赖**
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **访问应用**
打开浏览器访问 `http://localhost:5173`

### 构建部署

1. **构建项目**
```bash
npm run build
```

2. **部署到Netlify**
- 将项目推送到GitHub
- 在Netlify中连接GitHub仓库
- 设置构建命令：`npm run build`
- 设置发布目录：`dist`
- 点击部署即可

## 📖 使用指南

### 1. 基础信息填写
- 在左侧编辑区填写个人基本信息
- 右侧预览区会实时显示格式化后的简历

### 2. 技能管理
- 输入技能类别和具体内容
- 点击"添加技能"按钮
- 可随时删除不需要的技能项

### 3. 项目经验
- 填写项目的详细信息
- 支持多个项目管理
- 自动生成专业的项目展示格式

### 4. 版本管理
- **保存版本**：为不同岗位保存专门的简历版本
- **切换版本**：快速在不同版本间切换
- **导出备份**：将所有版本导出为JSON文件
- **导入备份**：从备份文件恢复数据

### 5. PDF导出
- 点击"导出PDF"按钮
- 自动生成A4格式的PDF文件
- 保持与预览区完全一致的格式

## 🛠️ 技术栈

- **前端框架**：Vanilla JavaScript + Vite
- **样式**：原生CSS3
- **PDF导出**：html2pdf.js
- **数据存储**：localStorage
- **部署平台**：Netlify

## 💡 使用技巧

### 秋招优化建议
1. **技能排序**：根据目标岗位JD调整技能顺序
2. **数据量化**：项目成果用具体数字描述（如"提升30%"、"日均50人"）
3. **版本命名**：建议用"岗位-公司"格式命名版本（如"前端-字节"）
4. **定期备份**：重要数据及时导出备份，避免浏览器清理导致丢失

### 性能优化
- 本地存储容量约5MB，建议定期清理无用版本
- 项目描述控制在合理长度，避免PDF导出时格式错乱
- 定期导出备份，防止数据丢失

## 🔧 自定义配置

### 修改简历样式
在 `index.html` 的 `<style>` 标签中修改CSS样式，预览区会自动应用新样式。

### 添加新字段
1. 在 `index.html` 中添加对应的表单元素
2. 在 `src/main.js` 中更新数据结构和事件监听
3. 在 `updatePreview()` 函数中添加渲染逻辑

## 🐛 常见问题

**Q: PDF导出后格式错乱怎么办？**
A: 检查预览区内容是否超出A4纸张范围，适当调整内容长度。

**Q: 版本切换后数据丢失？**
A: 确保在切换前保存当前版本，或使用导出备份功能。

**Q: 移动端显示异常？**
A: 项目已适配移动端，如有问题请检查浏览器兼容性。
